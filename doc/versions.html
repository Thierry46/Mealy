Mealy project - Versions


Object

This file describes the evolution and content of Mealy project

Version list

3.0 : 10/2/2021

  • porting from python2 to python 3
  • convert sources to utf-8 format
  • deal with french accents in texts to process
  • Ignore chars the are not in internal alphabet
  • Simplify properties file name : Mealy.properties -> Mealy.ini
  • add property MealyMachine/allowedCharsExt to allow new chars in accepted alphabet
  • add new test cases
  • properties for Test cases

2.0 : 22/5/2015

  • Enable hiding message in PNG image with steganography.
    This feature is disabled if PIL/Image and stepic modules are not installed.

v1.2 : 19/5/2015

  • Gnu GPL v3 license for the project
  • Documentation in doc directory
  • Possibility to enter alphanumeric optional key directly
  • Correct bug in GUI suppressing last char of coding messages.

v1.1 : 15/5/2015

  • Mono windows GUI : frame are stacked like a pile of card : MealyGUI.py, CardFrame.py
  • New classes for each frame : KeysFrame.py, CypherFrame.py, ResultFrame.py
  • Application properties are stored in Mealy.properties text file

v1.0 : 11/5/2015

  • object version
  • Multi windows GUI  based on tkinter : MealyGUI.py
  • i18n : Internationalization based on GNU gettext :
    French translation in sub directory: locale/fr/LC_MESSAGES/messages.po
  • program launcher for GUI and batch mode : run_Mealy.py

v0.1 - 4/5/2015

  • Mealy module ok, procedural version : Mealy.py
  • unit tests written : test_Mealy.py
